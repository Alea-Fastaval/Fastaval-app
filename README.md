# Fastaval App

A Flutter project for Alea's Fastaval

## Commands

| Command            | Descriptions                                                                |
| ------------------ | --------------------------------------------------------------------------- |
| `flutter run`      | Starts the project in a local browser                                       |
| `flutter gen-l10n` | Generates the translationfiles located in `.dart_tool/flutter_gen/gen_l10n` |
